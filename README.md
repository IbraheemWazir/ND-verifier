# ND-verifier
A natural deduction proof verifier in haskell

## Executing the verifier
Given the path to your proof file (PATH_TO_FILE), type the following in the 
```
  ./Main PATH_TO_FILE
```

## Rules
### And Introduction
``` 
q (somehow)
p (somehow)

p and q
```
### And Elimination
```
p and q (somehow)

p
q
```
